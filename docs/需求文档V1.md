# 成本管理看板 - 需求文档（优化版）

> 本文档在原版基础上，针对**样式设计、信息架构、交互体验**进行了全面增强，以满足政府 IT 服务商对数据可视化与专业性的更高要求。

---

## 一、项目背景

### 1.1 业务背景
- **公司类型**：政府 IT 服务商
- **服务对象**：省级、市级、地市州级甲方
- **部门规模**：约 25 人
- **项目数量**：约 30 个项目（对应成都 23 区县 + 市本级 + 省级）

### 1.2 业务模式
- 销售按市场区域划分，年初设定签单目标
- 项目成本构成：
  - 内部人力（固定成本）
  - 外采人力（可变成本）
  - 商务成本
  - 其他成本

---

## 二、用户角色

### 2.1 老板
- **核心需求**：查看总览数据和统计看板
- **关注指标**：
  - 年度收入/成本/利润（计划 vs 实际）
  - 成本分类明细
  - 项目整体健康度

### 2.2 项目经理
- **核心需求**：跟进项目进程和成本花费，保证利润率
- **关注指标**：
  - 单个项目签约金额、成本、利润率
  - 项目进度
  - 预估总成本
  - 成本预警（是否超支）

### 2.3 销售
- **核心需求**：签单目标完成情况、各项目利润率
- **关注指标**：
  - 目标完成率
  - 负责项目的利润率
  - 项目数量

## 三、整体布局与导航

### 3.1 页面结构
```
┌─────────────────────────────────────────────────────────────┐
│ 左侧导航栏（深蓝 #1a2740）  │       顶部工具栏                 │
│  - 品牌区（Logo + 系统名）  ├─────────────────────────────────┤
│  - 总览看板（图标+文字）    │ [更新时间] [导出] [用户头像]    │
│  - 项目维度                 │                                 │
│  - 销售维度                 │        内容区域（#f5f6fa）       │
│  [收起按钮]                 │        24px 栅格系统             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 导航栏样式规范
- 背景色 `#1a2740`，宽度 240px（收起时 64px）
- 菜单项高度 48px，圆角 8px，左侧选中态增加 4px 蓝色条（#4080FF）
- 图标使用统一线条风格，hover 时背景 `rgba(255,255,255,0.1)`

### 3.3 顶部工具栏
- 右对齐布局
- **数据更新时间**：格式 `YYYY-MM-DD HH:MM`，带刷新图标
- **快捷操作**：导出截图（PNG）、刷新数据（重新加载 mock）
- **用户头像**：圆形，下拉菜单显示角色切换（老板/项目经理/销售），用于预览不同视角

### 3.4 内容区通用规范
- 栅格系统：24 列，间距 24px
- 卡片：圆角 12px，背景白色，默认阴影 `0 2px 8px rgba(0,0,0,0.04)`，hover 阴影 `0 4px 12px rgba(0,0,0,0.08)`
- 内边距：卡片内容 `20px`，标题栏 `16px 20px`
- 字体：系统默认（Inter / SF Pro / Roboto），标题 20px/600，卡片标题 16px/500，正文 14px/400，辅助文字 12px/400
- 过渡动效：所有交互（hover、展开、排序）过渡时间 0.2s，缓动 ease

## 四、功能模块详细设计

### 4.1 部门总览看板（Dashboard）

#### 4.1.1 核心指标卡片
采用 **三列布局**（收入/成本/利润），每个卡片结构：

| 元素 | 样式要求 |
|------|----------|
| 指标名称 | 14px 次要文字色（#666） |
| 主数值 | 32px 加粗，主文字色（#1a2740） |
| 对比信息 | 计划值（12px 灰），完成率（12px 带箭头图标） |
| 进度条 | 高度 8px，圆角 4px，背景 #E5E7EB，填充色动态：<br>≤80% 橙红渐变，80%~100% 蓝色渐变，≥100% 绿色渐变 |
| 趋势标识 | 实际 vs 计划差异：↑（绿色） / ↓（红色）+ 百分比 |

**示例卡片**：
```
年度收入（万元）
      ¥ 2,580
计划 ¥ 2,400  完成率 107.5% ↑7.5%
████████████░░░░ 87%
```

#### 4.1.2 项目概况
- 卡片内横向排列四个指标（总数 / 进行中 / 已签约 / 待评估 / 已完成）
- 数字使用 24px 加粗，标签 14px 次要色
- 可选：迷你进度条展示各状态占比

#### 4.1.3 收入对比图
- 柱状图：计划收入（浅灰 #E5E7EB）、实际收入（主色 #4080FF 带轻微渐变）
- 坐标轴颜色 #E5E7EB，轴线细 1px
- 柱顶显示数据标签
- Tooltip：白色背景、圆角 8px、阴影、数值加粗
- 支持图例点击显示/隐藏对应数据

#### 4.1.4 成本结构图
- 采用**环形图**（中间显示总成本金额），比普通饼图更节省空间
- 配色方案：内部人力 #4080FF，外采人力 #14C9C9，商务成本 #FAC858，其他成本 #EE6666
- 图例置于右侧或底部，支持点击高亮对应扇形
- 悬停显示具体金额与占比

#### 4.1.5 成本分类明细表
表格列：
| 成本类型 | 计划金额 | 实际金额 | 差异 | 完成率 | 进度条 |
|----------|----------|----------|------|--------|--------|

**样式要求**：
- 表头固定，支持横向滚动
- 差异值红色（超支）或绿色（节约），带 +/- 号
- 进度条同核心卡片规范
- 行高 56px，悬停背景 #F9FAFB

#### 4.1.6 利润率对比
- 使用仪表盘或双进度条形式，清晰展示计划利润率 vs 实际利润率
- 实际利润率低于计划时，用红色警告标识

### 4.2 项目维度（ProjectView）

#### 4.2.1 区县项目金额分布
- 横向柱状图，显示前 12 个区县的签约金额
- 项目名称左对齐，金额右对齐，柱条颜色 #4080FF
- 前 3 名增加 🏆 图标
- hover 显示详细金额

#### 4.2.2 项目状态筛选
- 采用 **Segmented 控件**：全部 / 进行中 / 已签约 / 待评估 / 已完成
- 选中态：蓝色文字 + 底部蓝色下划线
- 每个状态旁显示数量徽章（如“进行中 12”）

#### 4.2.3 项目列表表格
| 字段 | 说明 | 样式增强 |
|------|------|----------|
| 项目名称 | 含区县信息 | 主文字加粗，次级区县信息 12px 灰色 |
| 状态 | Tag 显示 | 颜色区分（进行中-蓝，已签约-绿，待评估-橙，已完成-灰） |
| 签约金额 | actualAmount | 可排序，右对齐 |
| 成本 | actualCost | 可排序，右对齐 |
| 利润率 | 计算字段 | 可排序，根据与计划利润率对比增加背景色：<br>≥预期：浅绿背景 #E8F5E9，↑图标<br><预期：浅红背景 #FFEBEE，↓图标 |
| 进度 | progress | 进度条（高6px，圆角，蓝色填充）+ 百分比显示在右侧 |
| 预估总成本 | estimatedTotalCost | 右对齐，当预估 > 实际成本*1.2 时显示预警图标 ⚠️ |
| 销售/项目经理 | 双行显示 | 销售（14px）、项目经理（12px 灰色），缺失时显示“待分配” |

**交互**：
- 表头可排序字段带排序图标（↑↓）
- 支持全局搜索（项目名称/区县）
- 行悬停显示操作按钮（查看详情）

### 4.3 销售维度（SalesView）

#### 4.3.1 顶部统计卡片
- 横向排列五个指标：总计划收入、总实际收入、完成率、平均利润率、总项目数
- 完成率卡片增加趋势箭头（同比去年）
- 卡片背景使用微渐变，提升层次感

#### 4.3.2 目标完成率图
- 柱状图：每个销售的目标完成率
- 颜色规则：
  - ≥ 100%：绿色渐变
  - ≥ 80%：黄色渐变
  - < 80%：红色渐变
- 柱顶显示完成率百分比
- 图例显示“目标完成率”

#### 4.3.3 利润率对比图
- 分组柱状图：计划利润率 vs 实际利润率
- 计划利润率柱浅灰色，实际利润率柱主色
- 数据标签显示利润率值

#### 4.3.4 销售明细表
| 字段 | 说明 | 样式增强 |
|------|------|----------|
| 销售 | 姓名 | 加粗 |
| 计划收入 | planRevenue | 右对齐 |
| 实际收入 | actualRevenue | 右对齐，可排序 |
| 目标完成率 | 计算字段 | 进度条（高8px，圆角）+ 百分比，动态颜色同上 |
| 利润率 | 实际/目标 | 带颜色背景和趋势图标 |
| 项目数 | projectCount | 居中，点击可跳转至该项目列表（下钻） |

**角色视角**：
- 当用户为销售时，默认只显示本人的数据，但可切换查看团队
- 顶部卡片显示个人目标完成率、负责项目数

## 五、交互与反馈

### 5.1 加载状态
- 骨架屏（卡片、图表占位）替代传统 loading 动画

### 5.2 错误提示
- 数据加载失败时，卡片内显示“数据加载失败，点击重试”按钮

### 5.3 空状态
- 表格无数据时，显示“暂无数据”图标 + 提示文字

### 5.4 响应式适配
- 侧边栏收起时，内容区自动延展
- 屏幕宽度 < 1024px 时：
  - 核心指标卡片改为 2x2 网格
  - 图表改为堆叠布局
  - 表格支持左右滑动，增加“查看详情”抽屉
