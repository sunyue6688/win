## 设计决策记录

1. **成本占比图表形态**：左环状图（各类型占比）+ 右水平条形图（计划vs实际对比）
2. **成本卡片基准值**：计划成本/签约收入，基准30%，超出红色+⚠️，否则绿色
3. **贡献值系数**：暂定1%，后续可配置
4. **图表配色**：统一蓝色系，避免杂乱多色
