# 成本管理看板 - UI 设计规范（优化版）

> 本文档从产品需求文档中提取 UI/UX 相关规范，并进行补充优化，旨在为前端开发提供清晰、可执行的设计标准。

---

## 一、整体布局

### 1.1 页面结构
```
┌─────────────────────────────────────────────────────────────┐
│ 左侧导航栏（深蓝 #1a2740）  │       顶部工具栏                 │
│  - 品牌区（Logo + 系统名）  ├─────────────────────────────────┤
│  - 总览看板（图标+文字）    │ [更新时间] [导出] [用户头像]    │
│  - 项目维度                 │                                 │
│  - 销售维度                 │        内容区域（#F8FAFC）       │
│  [收起按钮]                 │        24px 栅格系统             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 布局规范
| 元素 | 规范 |
|------|------|
| 侧边栏宽度 | 展开 240px，收起 64px |
| 内容区内边距 | 24px |
| 卡片间距 | 24px |
| 栅格系统 | 24 列，基于 8px 网格 |

## 二、色彩系统

### 2.1 主色与辅助色
| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 侧边栏背景 | #1a2740 | 深蓝主色 |
| 内容背景 | #F8FAFC | 全局背景 |
| 主色-蓝 | #4080FF | 按钮、选中态、图表主色 |
| 辅助色-青 | #14C9C9 | 辅助强调、渐变 |
| 警告色-黄 | #FAC858 | 商务费用、接近阈值 |
| 危险色-红 | #EE6666 | 超支、不达标 |
| 成功色-绿 | #3DC779 | 达标 |

### 2.2 文字颜色
| 类型 | 颜色值 |
|------|--------|
| 主要文字 | #1a2740 |
| 次要文字 | #666 |
| 辅助/占位 | #999 |

### 2.3 边框与分割线
- 默认边框：`#E5E7EB`
- 表格分割线：`#F0F2F5`
- 卡片阴影：`0 2px 12px rgba(0,0,0,0.04)`（默认），hover 加深至 `0 4px 16px rgba(0,0,0,0.08)`

## 三、组件样式规范

### 3.1 卡片
- 圆角：`16px`
- 内边距：`20px`
- 背景：白色
- 标题：`16px/500`，主色

### 3.2 按钮
- 主要按钮：背景 `#4080FF`，圆角 `8px`，内边距 `8px 16px`
- 次要按钮：边框 `#E5E7EB`，文字 `#666`
- 悬停：透明度或颜色加深 10%

### 3.3 表格
- 表头背景：`#F9FAFB`
- 表头文字：`14px/500`，次要色
- 单元格：`14px/400`，行高 `56px`
- 悬停行背景：`#F9FAFB`
- 固定表头，支持横向滚动

### 3.4 标签（Tag）
| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 进行中 | #E8F2FF | #4080FF |
| 已签约 | #E8F5E9 | #3DC779 |
| 待评估 | #FFF4E5 | #FAC858 |
| 已完成 | #F2F4F8 | #9CA3AF |

### 3.5 进度条
- 高度：`8px`
- 圆角：`4px`
- 背景：`#E5E7EB`
- 填充色：
  - 达标（完成率≥100%）：绿色渐变
  - 正常（80%~100%）：蓝色渐变
  - 预警（<80%）：橙红渐变
- 百分比标签：右侧，`12px`

### 3.6 筛选组件（Segmented）
- 选项间距：`8px`
- 选中态：蓝色文字 + 底部蓝色下划线（宽度 24px，高度 2px）
- 徽章：选中态显示数量，灰色圆点，右对齐

### 3.7 抽屉（Drawer）
- 宽度：`480px`
- 从右侧滑出
- 遮罩半透明：`rgba(0,0,0,0.4)`
- 内边距：`20px`

## 四、图表设计

### 4.1 通用配置
- 字体：`12px`，次要色
- 坐标轴线：`#E5E7EB`，线宽 `1px`
- 网格线：`#F0F2F5`，线宽 `1px`
- Tooltip：白色背景，圆角 `8px`，阴影，数值加粗

### 4.2 环形图
- 中心显示总成本金额（`20px` 加粗）
- 配色：固定成本 `#E5E7EB`，外采人力 `#4080FF`，商务费用 `#FAC858`，其他 `#EE6666`
- 图例置于右侧或底部，支持点击显示/隐藏

### 4.3 柱状图
- 计划收入柱：浅灰 `#E5E7EB`
- 实际收入柱：主色渐变 `#4080FF → #14C9C9`
- 柱顶数据标签：白色背景，圆角 `4px`，内边距 `2px 4px`

### 4.4 折线图
- 实际利润率折线：蓝色
- 计划利润率折线：浅灰虚线
- 低于计划时折线变为红色，并标注预警点（红色圆点，带 tooltip）

## 五、动效与过渡

| 元素 | 动效 |
|------|------|
| 页面切换 | 淡入淡出，`opacity` 过渡，时长 `0.2s` |
| 卡片 | 依次出现（stagger 0.05s），轻微上浮（translateY -4px） |
| 图表 | ECharts 默认入场动画（扇形展开、柱增长） |
| 表格行悬停 | 背景色 `#F9FAFB`，过渡 `0.2s` |
| 按钮/卡片悬停 | 阴影加深，过渡 `0.2s` |

## 六、响应式适配

### 6.1 断点
- 桌面：≥1280px
- 平板：1024px ~ 1279px
- 移动：<1024px

### 6.2 平板适配
- 侧边栏固定收起状态（64px）
- 核心指标卡片改为 2x2 网格
- 图表堆叠排列（环形图在上，柱状图在下）
- 表格允许横向滚动

### 6.3 移动适配（未来）
- 隐藏侧边栏，改为底部标签栏
- 卡片单列布局
- 表格仅展示关键字段，详情通过抽屉查看

## 七、图标使用

统一使用 Semi Design 内置图标库 `@douyinfe/semi-icons`，常用图标：
- 预警：`IconWarning`
- 上升/下降：`IconArrowUp` / `IconArrowDown`
- 搜索：`IconSearch`
- 筛选：`IconFilter`
- 详情：`IconMore`
- 导出：`IconDownload`
- 刷新：`IconRefresh`

---

## 八、开发交付要求

1. 所有样式遵循上述规范，避免硬编码颜色值，使用 CSS 变量或 Semi Design 主题令牌。
2. 图表必须包含入场动画和悬停交互。
3. 表格排序、搜索、筛选等交互必须平滑无卡顿。
4. 响应式适配至少覆盖平板尺寸。
5. 代码注释清晰，组件可复用。
