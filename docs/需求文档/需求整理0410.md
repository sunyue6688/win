# 二、需求目标&计算规则
目前老板给我提供了她已经设计好的Excel表格，这里面的内容是她最关注的点，也是我们看板中需要展示的内容。（统计时间是截止到2026年4月1日）  
我们目前对成本的结构有两种维度：

纬度一：外部成本管理视角（结合需求文档V8.md查看）
1. 总成本=内部成本+外部成本
2. 内部成本=部门人员投入本部门项目人力成本+部门日志向项目分摊成本
3. 外部成本=交付成本（外）+商务成本
4. 交付成本（外）=交付成本（外采成本）+交付成本（外部门成本）=项目研发外采付款+项目运营外采付款+外部门投入本部门项目人力成本+外包人员成本1
5. 交付成本（外采成本）=项目研发外采付款+项目运营外采付款+外包人员成本1
6. 交付成本（外部门成本）=外部门投入本部门项目人力成本
7. 商务成本=商务成本（外采成本）+商务成本（集采成本）=项目商务外采付款+商务其他成本+外包人员成本2
8. 商务成本（外采成本）=项目商务外采付款+外包人员成本2
9. 商务成本（集采成本）=商务其他成本

纬度二：项目管理视角
1. 管理成本=部门人员投入本部门日志+外部门投入本部门日志
2. 交付成本=部门人员投入本部门项目人力成本+部门日志向项目分摊成本+交付成本（外）
3. 商务成本=项目商务外采付款+商务其他成本+外包人员成本2

每个字段的计算方式及来源：
1. 部门人员投入本部门项目人力成本：内部人员填写的项目日志+商务日志（部门名称：社会工作业务部）（应该需要用到内部人员名单、日志标准、商机日志明细汇总表、项目日志明细汇总）；
2. 部门日志向项目分摊成本：部门日志中，需要分摊向项目的部分（需要用到日志标准（根据字段”部门日志是否分摊“=是进行计算））；
3. 外部门投入本部门项目人力成本：非内部人员填写的项目日志+商务日志；
4. 项目研发外采付款、项目运营外采付款、项目商务外采付款均来源【项目经理表】社工业务项目进展台账.xlsx中外采类型匹配计算值；
5. 外包人员成本1、外包人员成本2、商务其他成本均来源超级管理员填写；
6. 部门人员投入本部门日志、外部门投入本部门日志：部门日志表中可以查看。
